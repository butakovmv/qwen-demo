# Модуль app

## Назначение
Первичная точка входа, настройки Spring Boot приложения. Инициализирует контекст и запускает сервер.

## Содержание
- `OtusApplication.kt` — главный класс с `main()` и `@SpringBootApplication`
- `application.yml` — настройки приложения (порт, профили)

## Связи
Зависит от всех модулей-адаптеров (`web-api`, `usecase`) для сборки полного контекста.
Не содержит бизнес-логики — только конфигурация и запуск.

## Сборка
Собирается в executable jar через `./gradlew :app:bootJar`.
Запускается через `./gradlew :app:bootRun` или `java -jar app.jar`.

## Тесты
Содержит тесты контекста приложения — проверка корректности сборки Spring DI.
